name: Concise
description: Tight single-page format — ideal for candidates with extensive experience
---

LAYOUT INSTRUCTIONS:
- Header: Name + essential contact info only (email, LinkedIn, location) — no phone if space is tight
- Sections in this order: Summary (2 sentences max), Experience, Skills, Education
- Summary: punchy 2-sentence professional pitch only
- Experience: maximum 3 most recent/relevant roles; 2-3 bullets each (no more)
- Each bullet: one sentence, result-first format ("Reduced build time 40% by...")
- Skills: single compact line per category, abbreviate where obvious
- Education: one line per degree (Degree, Institution, Year)
- Goal: fit everything comfortably on one page
- Ruthlessly cut anything not directly relevant to the target job
